ngToast

ngToast is a simple Angular provider for toast notifications.

Demo

Create a sample toast

Usage

Download ngToast manually or install with bower:

bower install ngtoast

Include ngToast resource files along with the Bootstrap CSS (only the Alerts component is used as style base, so you don't have to include complete CSS):

<link rel="stylesheet" href="lib/bootstrap/bootstrap.css">
<link rel="stylesheet" href="lib/ngToast/ngToast.css">

<script src="lib/ngToast/ngToast.min.js"></script>

Include ngToast as a dependency in your app module:

var app = angular.module('myApp', ['ngToast']);

Place ng-toast element into your HTML:

<body>
  <ng-toast></ng-toast>
  ...
</body>

Inject ngToast provider in your controller:

app.controller($scope, ngToast) {...};

API

// create a toast:
ngToast.create('A toast message...');

// clear specific toast:
var msg = ngToast.create({
  content: 'Another message as <a href="#" class="">HTML</a>'
});
ngToast.dismiss(msg);

// clear all toasts:
ngToast.dismiss();

Settings

Property Default Description
content Content of the toast message as string (HTML is supported).
class 'success' Class name to be added on toast message with 'alert-' prefix. See Bootstrap Alerts for possible options.
dismissOnTimeout true Allows toast messages to be removed automatically after a specified time
timeout 4000 Wait time for removal of created toast message
dismissButton false Appends specified close button on toast message
dismissButtonHtml '&times;' HTML of close button to append
dismissOnClick true Allows toasts messages to be removed on mouse click
horizontalPosition* right Horizontal position of toast messages: 'right', 'left' or 'center'
verticalPosition* top Vertical position of toast messages: 'top' or 'bottom'
maxNumber* 0 Maximum number of toast messages to display. (0 = no limit)
* Available only at configuration level. Rest of setting properties can be overwritten at individual toast message.
